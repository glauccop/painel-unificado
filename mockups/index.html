▱Painel Unificado · CAIXA — 3 ideias para a aba Esteira

Mesmo shell do painel (seletor de Área/SIGLA · Minha Carteira · Por SIGLA · DomainCards dos módulos ServiceNow). O que muda entre os três é só como a aba Esteira mostra a rastreabilidade ponta a ponta entre os módulos. Abra, compare e escolha um — depois evoluímos o escolhido para o app real com dados da instância.

⚠️ Protótipos com dados de exemplo. 3 SIGLAs fictícias (SIAME_RJ, SIANA, SIAOC) e registros mock — só para decidir o desenho. A integração com dados reais (ITSM, Ágil, SPM, EA/APM) e a definição da fonte de SIGLA vêm na fase seguinte.

Ideia A

Trilhos de cadeia

Cada rastreabilidade vira um trilho horizontal Demanda → Requisito → Story → Teste → Operação. Lê-se como uma linha do tempo; ótimo para seguir um item ponta a ponta.

Foco: acompanhar uma cadeia do começo ao fim. Abrir protótipo A →
Ideia B

Kanban de fases + faixa

Cinco colunas por fase do ciclo, com linhas conectando os cards entre os módulos. Clicar num card acende toda a linhagem e monta a faixa de rastreabilidade embaixo.

Foco: visão de pipeline + drill numa cadeia sob demanda. Abrir protótipo B →
───
Ideia C

Grafo de rastreabilidade

Mapa node-link conectando registros entre os módulos. Evidencia a teia inteira e destaca lacunas — requisitos e itens sem caso de teste ficam marcados em vermelho.

Foco: rede de relações e cobertura (o que está “órfão”). Abrir protótipo C →

O que é igual nos três (o shell do painel)

  • Abre em Minha Carteira — os itens atribuídos a você, de todos os módulos.
  • Trocar a SIGLA no topo leva automaticamente à visão Por SIGLA daquele sistema.
  • Os 6 módulos (ITSM · Ágil · SPM · Requisitos · Testes · EA/APM) saem do mesmo DomainCard.
  • A aba Esteira mostra a rastreabilidade ponta a ponta entre os módulos.
